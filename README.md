# AI Program Manager — 14-Week Learning Plan

**A structured, hands-on learning journey to master AI as a Program Manager or Agile Coach.**

This repository contains a complete **14-week learning plan** designed specifically for Program Managers, Agile Coaches, and technical leaders who want to become proficient in AI and lead AI initiatives effectively.

---

## 🌟 What You'll Get

- **14-week structured curriculum** focused on practical AI skills
- Full-code **hands-on projects** you can run locally
- Free learning resources (videos, articles, tools)
- Progress tracker (saves in your browser)
- Real-world project examples from my AI portfolio

1. **Visit the Learning Plan**  
   👉 **[Open the Interactive Learning Plan](https://bhatiaaman.github.io/Ai-Learning-Plan/)**

   All projects are available in the companion repository:

   - Go through one week at a time

   - Build the weekly project
   - Track your progress using the built-in tracker
   - Repeat for 14 weeks
